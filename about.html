This serves as an APPROXIMATION of the flight path that the SpaceX booster will take over the three months preceding its impact with the moon. The moons orbit was defined using and ephemeris generated using JPL's Horizon system at ssd.jpl.nasa.gov, and the booster flight path was defined using an ephemeris extracted from this tool, created by Bill Gray, who discovered that this event will take place.
